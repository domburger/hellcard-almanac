About

The Hellcard Almanac is a community-built reference for Hellcard, a cooperative roguelike deckbuilder where up to three players descend into procedurally generated dungeons. This site catalogues every card, artifact, enemy, and location in the game so you can plan upgrades, theorycraft builds, and look up mechanics between runs.

Features

  • Searchable card database with evolution tree and list views
  • Deck builder with local save/load
  • Import and export deck codes
  • Enemy stats and ability reference
  • Location options and costs
  • Starter artifact comparisons
  • Mobile-friendly dark theme
